About

This document describes how to build jamod from the source package.

Invoking the build

jamod's build is based on ant and integrated in the repository.

Unix platforms
The script that will invoke ant is build.sh.
Windows platforms
There is a build.bat batch script which will invoke ant.

Both scripts are courtesy of the ant project; the following snippets show how to invoke them: 

[Fangorn:~/development/java/jamod] wimpi$ ./build.sh [ant options] <targetname>
C:\jamod\> build [ant options] <targetname>
Note
You can also use ant directly if you have it installed for use with many projects. The build script is build.xml.
In either case you might want to change settings loaded from the build.properties file in the working directory, or the .build.properties file in your user home directory.

To see the build targets and the default target, you should invoke ant with the option -projecthelp

Note
All build products including the docs and the distributions will be written to the build directory.
